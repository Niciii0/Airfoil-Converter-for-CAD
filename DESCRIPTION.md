# GitHub Description

Airfoil Converter V6.2 is a Windows desktop wing-design utility for browsing airfoil coordinate files, building editable wing station layouts, blending airfoils, previewing geometry in 3D, running XFOIL polar analysis, and exporting CAD/simulation-ready geometry.

## Short Repository Description

Tkinter-based airfoil and wing station design tool with 3D preview, XFOIL polars, airfoil blending, cleanup tools, and CAD/mesh exports.

## Release Summary

Airfoil Converter V6.2 adds a more complete wing-design workflow:

- adjustable GUI layout
- checkbox station selection
- two-station airfoil blending
- resizable sidebar and station list
- 3D preview with zoom and view controls
- XFOIL polar computation and scoring
- airfoil geometry analysis and cleanup
- OBJ/STL wing mesh export
- standalone Windows build support
